# Disruptive Signals Detector

A repository for detecting disruptive signals in operational or event-driven data streams. This project is designed to identify anomalies, trigger alerts, and surface potential disruptions using a combination of data processing and signal detection techniques.

## Features

- Signal ingestion and preprocessing
- Alert generation for disruptive patterns

## Installation

1. Clone the repository

   ```bash
   git clone <repository-url>
   cd disruptive_signals_detector
   ```

2. Install dependencies

   ```bash
   pip install -r requirements.txt
   ```

## Notebooks

For data preparation notebooks, check [notebooks/explore_clean_data.ipynb][notebooks/explore_clean_data.ipynb], while for analysis see [notebooks/aviation_disruption_detector.ipynb](notebooks/aviation_disruption_detector.ipynb).
